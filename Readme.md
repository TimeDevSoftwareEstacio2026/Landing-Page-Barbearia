# 💈 Barberia Landing Page

Projeto desenvolvido como parte de um trabalho universitário, com o objetivo de criar uma landing page para uma barbearia utilizando **HTML, CSS e JavaScript puro (vanilla)**.

## 📌 Objetivo

Criar uma página moderna, responsiva e visualmente atraente que apresente os principais serviços de uma barbearia, incluindo informações como:

- Serviços oferecidos
- Tabela de preços
- Horários de funcionamento
- Localização
- Seção visual (galeria)

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (básico)
- Metodologia Scrum (organização em sprints semanais)

## 📂 Estrutura do Projeto
│
├── index.html

## 🧩 Funcionalidades

- ✅ Estrutura HTML semântica
- ✅ Header com navegação
- ✅ Seção de horários
- ✅ Tabela de preços
- ✅ Seção de localização com mapa
- ✅ Seção visual (imagens da barbearia)
- 🔄 Responsividade (em desenvolvimento)
- 🔄 Interações com JavaScript (básicas)

## 📅 Organização (Scrum)

O projeto está sendo desenvolvido em sprints semanais, com divisão de tarefas entre os integrantes:

- Estrutura HTML geral e localização
- Header (topo)
- Tabela de preços
- Seção de horários
- Seção visual
- Footer
